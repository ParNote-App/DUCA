#### Prerequisites

PHP v5.6+ <br />
Docker <br />
Composer

## Development

#### Getting Started

Clone this repository.

```bash
git clone --recursive https://github.com/ParNote-App/DUCA.git
cd DUCA
```

##### Install Dependencies:

```bash
composer install
```

##### Run With Docker:

```bash
docker-compose up
```
